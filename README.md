# node-diagram-viewer

最小の **ASCIIマインドマップ（Flow × Layer）** を起点に、設計書テンプレを自動生成する実験リポジトリ。  
- 目的: 構造（Layer）と流れ（Flow）を並列に可視化  
- キーファイル: \origin_vector.yaml\（起点ベクトル）

## クイック
\\\pwsh
git clone https://github.com/takafumiooshita/node-diagram-viewer.git
cd node-diagram-viewer
